# Tweets-Sentiment-analysis-System
A web-based sentiment analysis application built using Python and Streamlit. This project analyzes Twitter data (offline dataset) and classifies tweets into Positive, Negative, and Neutral categories using Natural Language Processing (NLP).

# Project Overview
The Twitter Sentiment Analysis System helps in understanding public opinion by analyzing the emotional tone of tweets. It uses an offline dataset and TextBlob for sentiment classification and presents results through an interactive Streamlit dashboard.

This project is developed for academic and learning purposes.

# Objectives
Analyze sentiment of tweets using NLP techniques
Classify tweets into Positive, Negative, and Neutral categories
Provide an interactive and user-friendly dashboard
Visualize sentiment distribution using charts
Enable analysis of individual tweets and datasets

# Technologies Used
Python
Streamlit
Pandas
TextBlob
Matplotlib
Regular Expressions (re)

# Dataset
 Source: Kaggle Twitter Sentiment Dataset (Sentiment140)
 Format: CSV
 Size: 1.6 Million Tweets (sampled for performance)
 Type: Offline Dataset
 
# Features
✅ Single Tweet Sentiment Analysis
✅ Offline Dataset Analysis
✅ Interactive Dashboard
✅ Pie Chart Visualization
✅ CSV Export of Results
